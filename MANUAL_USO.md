# Manual de Uso - ATMOS App

Este manual explica como usar os principais modulos do app ATMOS.

## Login

1. Abra o app.
2. Informe email e senha.
3. Toque em entrar.

Cada lider deve usar seu proprio login.

## Tela Inicial

A tela inicial mostra os acessos principais pelo menu:

- Marcar Presenca
- Conectados
- Adicionar Visitante
- Cadastrar Adolescente
- Aniversariantes do mes
- Visitantes da Semana
- Relatorio Individual
- Relatorio Gerencial
- Relatorio Conectados
- Relatorio de Transicao

## Marcar Presenca

Use para registrar presenca nos eventos gerais.

1. Abra `Marcar Presenca`.
2. Escolha o evento.
3. Pesquise o adolescente, se necessario.
4. Toque no nome ou marque o checkbox.
5. O registro e salvo no evento do dia.

Eventos disponiveis:

- Culto Domingo Manha
- Culto Domingo Noite
- Conectadao
- Atmosfera
- Reuniao

Tambem e possivel exportar o PDF do dia pelo icone de PDF.

## Cadastrar Adolescente

Use quando um adolescente ainda nao existe no app.

1. Abra `Cadastrar Adolescente`.
2. Informe nome.
3. Informe data de nascimento, se tiver.
4. Informe telefone, se tiver.
5. Toque em salvar.

Antes de concluir, o app verifica nomes semelhantes. Se aparecer alguem parecido,
confira com cuidado para evitar cadastro duplicado.

## Adicionar Visitante

Use para registrar visitantes.

1. Abra `Adicionar Visitante`.
2. Informe nome.
3. Informe telefone e idade, se tiver.
4. Toque em salvar visitante.

## Visitantes da Semana

Mostra os visitantes registrados na semana atual.

Use para acompanhamento posterior e contato com quem visitou.

## Aniversariantes do Mes

Mostra aniversariantes do mes selecionado.

Ao tocar em um aniversariante, o app abre um cartao de aniversario personalizado
com:

- nome
- idade
- data
- versiculo
- mensagem em nome do ATMOS

O cartao pode ser compartilhado como imagem pelo celular quando o navegador
suportar o compartilhamento.

## Conectados

Use para gerenciar os grupos de discipulado.

### Abrir um grupo

1. Abra `Conectados`.
2. Toque no grupo desejado.

Cada grupo mostra nome, lider, membros e cor do Conectado.

### Registrar encontro

1. Abra o grupo.
2. Escolha a data do encontro.
3. Marque os adolescentes presentes.
4. No final, toque em `Confirmar Encontro`.
5. Confirme a acao.

Importante: o encontro so passa a valer para relatorios depois de tocar em
`Confirmar Encontro`.

### Corrigir presenca

Se precisar corrigir:

1. Abra o mesmo grupo.
2. Escolha a mesma data.
3. Ajuste os presentes.
4. Toque novamente em `Confirmar Encontro`.

### Adicionar adolescente ao Conectado

1. O adolescente precisa existir no cadastro principal.
2. Abra o Conectado.
3. Toque em `Adicionar adolescente existente`.
4. Escolha o adolescente.

### Transferir adolescente

1. Abra o Conectado atual.
2. No adolescente, abra as acoes.
3. Toque em `Transferir`.
4. Escolha o novo Conectado.

O historico anterior permanece salvo.

### Remover do Conectado

Remove apenas do grupo, nao do sistema.

1. Abra as acoes do adolescente.
2. Toque em `Remover do conectado`.
3. Confirme.

## Relatorio Individual

Use para acompanhar uma pessoa especifica.

1. Abra `Relatorio Individual`.
2. Pesquise o nome do adolescente.
3. Toque no resultado.

O relatorio mostra:

- dados do adolescente
- presencas
- faltas
- frequencia
- ultima presenca
- historico por evento
- Conectado atual
- presencas e faltas nos Conectados

## Relatorio Gerencial

Use para acompanhar o mes.

1. Abra `Relatorio Gerencial`.
2. Escolha o mes.
3. Veja os indicadores.

Ele ajuda a identificar adolescentes com baixa frequencia ou muitas faltas.

## Relatorio Conectados

Use para acompanhar os grupos de discipulado.

1. Abra `Relatorio Conectados`.
2. Escolha o mes.
3. Veja o panorama geral.
4. Toque em um grupo para abrir o detalhe.

No detalhe do grupo aparecem:

- encontros do mes
- presentes por encontro
- membros
- frequencia individual
- presenca/falta por data

Tambem e possivel exportar PDF mensal.

## Relatorio de Transicao

Mostra adolescentes ativos com 17 anos ou mais.

Use para acompanhar quem esta proximo de sair do departamento.

### Confirmar transicao

Quando o adolescente ja saiu do departamento:

1. Abra `Relatorio de Transicao`.
2. Toque em `Confirmar transicao`.
3. Leia a primeira confirmacao.
4. Confirme novamente na segunda tela.

Depois disso:

- ele deixa de aparecer nas listas ativas
- nao entra mais em marcacao de presenca
- nao aparece nos relatorios ativos
- o cadastro e historico continuam salvos

## Boas Praticas

- Sempre pesquise antes de cadastrar um adolescente.
- Confira nomes semelhantes antes de salvar novo cadastro.
- Nos Conectados, confirme o encontro apenas depois de revisar a lista.
- Use o Relatorio Individual para acompanhamento pastoral.
- Use o Relatorio de Transicao com cuidado, pois remove o adolescente das listas ativas.

## Suporte Operacional

Se algo aparecer errado:

1. Atualize a pagina.
2. Confira se esta logado com usuario correto.
3. Verifique se o adolescente esta ativo.
4. Em caso de duplicidade, comunique a administracao do app antes de cadastrar novamente.
